# Basics (Verilog / SystemVerilog)

This repository contains small **practice programs** written in **Verilog** and **SystemVerilog** to brush up HDL concepts.

## What this repo is for
- Refreshing core Verilog/SystemVerilog syntax
- Practicing digital design building blocks
- Writing simple testbenches and running simulations

## Typical practice topics
- Combinational logic
- Sequential logic
- FSM basics
- Testbenches

  ## Currently available code
- Frequency Divider (`Frequency_Divider.v`)

## Notes
- Modules may be independent and not follow a fixed structure (practice-oriented).
- I will keep adding examples as I revise concepts.
